希尔排序

基本思想

希尔排序(Shell Sort)是插入排序的一种。也称缩小增量排序，是直接插入排序算法的一种更高效的改进版本。希尔排序是非稳定排序算法。它的基本思想是先将整个待排元素序列分割成若干个子序列（由相隔某个“增量”的元素组成的）分别进行直接插入排序，然后依次缩减增量再进行排序，待整个序列中的元素基本有序（增量足够小）时，再对全体元素进行一次直接插入排序。因为直接插入排序在元素基本有序的情况下（接近最好情况），效率是很高的。

操作方法

  1. 先取一个小于N的整数h作为第一个增量，将所有序号相隔h的数组放在同一个组中；
  2. 在各组内进行直接插入排序；
  3. 取第二个增量h2\<h，重复上述的分组和排序；
  4. 直到所取的增量为1，即所有记录放在同一组中进行直接插入排序为止。

性质

性质 最差时间复杂度 最优时间复杂度 平均时间复杂度 所需辅助空间 稳定性
插入排序 已知最好的为O(n(logn)^2) O(n) 由步长而定 O(1) 不稳定

备注：
最差时间复杂度和平均时间复杂度 都是由步长序列的不同而不同
