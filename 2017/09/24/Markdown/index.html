Markdown语法的简要规则

标题

标题是每篇文章中常用的格式，在Markdown中，若一段文字被定义为标题，只要在这段文字前加#即可，总共六级标题建议在#后加一个空格，这是最标准的Markdown语法。

#一级标题

##二级标题

###三级标题

####四级标题

#####五级标题

######六级标题

列表

在Markdown下，列表的显示只需要在文字前面加上-或*即可变为无序列表，有序列表则直接在文字前1. 2. 3.符号要和文字之间添加一个字符的空格。

  • 真的是无序的
  • 竟然真的是无序的

  2. 有序列表
  3. 真的是有序列表

引用

如果你需要引用小段别处的句子，那面就要用到引用格式，只需在文本前加入>这种尖括号(大于号)即可

门前大桥下，游过一群虾，快来快来数一数，一只38~

粗体与斜体

Markdown的粗体和斜体也非常简单，用两个包含一段文本就是粗体的，用一个包含的一段文本就是斜体

表格

表格好累人啊~~
例如：
|tables |are |cool | |--------|:-----:|------:| |col 1 is|right-aligned|$1600| |col 2 is|centered|$1| |col 3 is|centered|$160|
这种语法生成的表格如下：

tables are cool
col 1 is right-aligned $1600
col 2 is centered $1
col 3 is centered $160

【回复】

【1】 Markdown可以利用反斜杠来插入一些在语法中有其他意义的符号，例如：你想要用星号加在文字旁边的方式来做出强调效果，你可以在星号的前面加上反斜杠： *literal asterisks *
Markdown支持以下这些符号前面加上反斜杠来帮助插入普通的符号：

\ 反斜杠
` 反引号
* 星号
_ 底线
{} 花括号
[] 方括号
() 括弧
# 井字号
\ + 加号
\ - 减号
. 英文句点
! 惊叹号
